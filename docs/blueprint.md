# **App Name**: Elite Agent

## Core Features:

- Asset Management: Store and manage all heavy assets such as 4K drone video and architectural photography using Supabase Storage.
- Bento Grid Gallery: Display property listings in a visually appealing Bento Grid, populated with images from Supabase Storage, using lazy loading and optimized WebP versions.
- Predictive Search: Implement a floating search bar with predictive suggestions powered by Supabase PostgreSQL database queries.
- MLS-AI Concierge: Integrate a conversational AI chatbot that acts as a property concierge tool by referencing property metadata stored in Supabase tables to answer user questions.
- Split-Hero Section: Use a split-screen hero section with a 4K drone video background fetched from Supabase Storage and a glassmorphic overlay for the value proposition.
- Mobile Quick Action Bar: Implement a bottom-anchored, sticky navigation bar with high-contrast buttons (Safety Orange) for key actions like 'Text an Agent'.

## Style Guidelines:

- Background: Matte Black (#121212) to provide a dark and luxurious backdrop.
- Sections: Deep Charcoal (#333333) for UI sections, adding depth.
- Accent (CTAs): Safety Orange (#FF6600) for high-commitment call-to-action buttons to ensure clear visibility and drive conversion.
- Accent (Icons/Active States): Brushed Gold (#D4A27A) for icons and active navigation states, providing a touch of elegance.
- Font Pairing: 'Playfair' (serif) for headlines to give an elegant, high-end feel; and 'PT Sans' (sans-serif) for body text, to ensure readability.
- Use minimalist icons with a 'Brushed Gold' finish to maintain a luxury aesthetic.
- Employ sharp, high-contrast shadows for cards and glassmorphism layers to create architectural depth and UI elevation, drawing inspiration from dramatic lighting.
- Subtle, smooth transitions and animations for UI elements to enhance the user experience without being distracting.
